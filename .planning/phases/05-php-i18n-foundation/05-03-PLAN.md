**D. Add switch_to_locale() logic to Plugin.php init callback:**
In the `init` hook callback (added in plan 05-01), AFTER `load_plugin_textdomain()`, add:

```php
$settings          = get_option( 'wmr_form_settings', array() );
$fallback_language = $settings['fallback_language'] ?? 'auto';

    switch_to_locale( 'de_DE' );
} elseif ( 'en' === $fallback_language ) {
    switch_to_locale( 'en_US' );
```
